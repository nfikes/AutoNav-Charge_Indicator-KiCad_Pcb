## Required first step

Before doing anything else in this repository, thoroughly read **`BEHAVIOR.md`** and **`RESEARCH.md`** in this directory.

- `BEHAVIOR.md` contains hard rules you must follow. **You may not modify it.**
- `RESEARCH.md` is your scratchpad of accumulated knowledge about this repository. **You may, and should, update it** as you learn more — keep entries grep-friendly (short, keyword-rich lines).

If either file is missing, stop and tell the user before proceeding.

If `RESEARCH.md` is empty (only the default scaffold), suggest to the user — once, on the first interaction in a session — that you run `/populate-research` to seed it before doing other substantive work. Do not run it without confirmation.
